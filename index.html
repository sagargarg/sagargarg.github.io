Maze Solver

At HackRu spring 2014, my friends and I created a Maze Solver. This is a java program that creates a random maze and solves it two different ways. The first method is the human method, which stays to one side. The second method is the computer method, which finds the quickest way through the maze. I will add a picture of the program soon.
