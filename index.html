LED Cube



Over the summer I built a 4x4x4 LED Cube. I built my LED Cube based on a tutorial from Tech Made Easy. The LED cube uses an Arduino Uno micro-controller to program the LEDs. The code that I used in the video along with additional pictures, can be viewed by clicking the link below. If you have any quesitons about the LED cube, please don't hesitate to contact me.





Maze Solver



At HackRu spring 2014, my friends and I created a Maze Solver. This is a java program that creates a random maze and solves it two different ways. The first method is the human method, which stays to one side, represented by the yellow line in the picture. The second method is the computer method, which finds the quickest way through the maze represented by the red line.




3-D molecule generator

At HackBCA spring 2014, my first hackathon, my friends and I created a 3-D molecule generator. The program would turn a 2-D lewis dot structure into a 3-D molecule. Since this was our first hackathon, we did not have much experience. Our project did not turn out as we hoped.
